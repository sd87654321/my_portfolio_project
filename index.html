Credit Card Fraud Detection

Overview

End-to-end fraud detection system for highly imbalanced transaction data - addressed the precision-recall tradeoff in financial fraud with SMOTE oversampling and comparative model evaluation focused on minimizing false negatives.

->

LR vs XGBoostTrain and Compare (Step 3)

AUC + Confusion MatrixRecall Optimized Output

Pipeline Code Snippet

from imblearn.over_sampling import SMOTE
from sklearn.model_selection import train_test_split
from xgboost import XGBClassifier
from sklearn.metrics import roc_auc_score, confusion_matrix

X_train, X_test, y_train, y_test = train_test_split(
    X, y, stratify=y, test_size=0.2, random_state=42
)
X_res, y_res = SMOTE().fit_resample(X_train, y_train)

clf = XGBClassifier(scale_pos_weight=10, eval_metric='aucpr')
clf.fit(X_res, y_res)
print("AUC:", roc_auc_score(y_test, clf.predict_proba(X_test)[:, 1]))
print(confusion_matrix(y_test, clf.predict(X_test)))

Approach

  • Handled extreme class imbalance using SMOTE oversampling and stratified data splitting.
  • Compared Logistic Regression and XGBoost on precision, recall, F1, and AUC-ROC.
  • Focused on confusion matrix analysis to minimize false negatives (missed fraud).
  • Built comprehensive EDA with Seaborn and Matplotlib to visualize fraud patterns.

import pandas as pd
import plotly.express as px

df = pd.read_csv('eia_renewable.csv', parse_dates=['date'])
df.dropna(subset=['consumption_btus'], inplace=True)

pivot = df.pivot_table(index='year', columns='source',
                       values='consumption_btus', aggfunc='sum')
df['renew_share'] = df['renewable'] / df['total_energy'] * 100

fig = px.line(pivot, title='US Renewable Energy by Source')
fig.show()

Approach
